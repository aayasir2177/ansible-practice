# Ansible Practice

This repo contains my ansible practice playbook created while learning Ansible from Learn Linux TV.

## Topics Covered

* Ad-hoc commands
* Playbooks and taskbooks
* Host vars
* Roles
* Package management
* Service management
* File management
* User management

## Purpose

The goal of this repo is to build Ansible skills through practical exercises.
